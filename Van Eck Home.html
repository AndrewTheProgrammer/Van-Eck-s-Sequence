Van Eck Sequence

Internal links: Home Python Generator First Million Numbers External links: Numberphile video OEIS entry
The Van Eck Sequence is a self-referential sequence. The first number is 0 and from then on, the next number is the number of times the last number last appeared. It was created by Jan Ritsema van Eck. The gennerator will refer to similar sequences with different starting numbers as "vaneckine".
